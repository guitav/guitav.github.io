Vim Commands

Delete

Command Action
x Delete current character
dw Delete current word
dd Delete current line
5dd Delete five lines (place any number before dd)
d$ Delete to end of line
d0 Delete to beginning of line
:1,.d Delete to beginning of file
:.,$d Delete to end of file
:1,$d Delete entire file

Miscellaneous

Command Action
:set tabstop=4 Sets tab length to 4
:set formatoptions-=cro Disables auto commenting
sed -n 22,44p filename > newfile Extracts from line 22 to 44 and inserts into new file
scp gvahdati@unix.ucsc.edu:/afs/cats.ucsc.edu/users/m/gvahdati/* . Copying files from remote system
1,122s/^/\/\/ Insert // from lines 1 to 122
